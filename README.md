# System Administrator Documentation Pack


This repository contains documentation, scripts, configuration samples, and diagrams demonstrating core competencies for a System Administrator I role. It includes examples related to:


- Windows OS troubleshooting
- Microsoft 365 / cloud-connected environments
- IPv4, DNS, DHCP diagnostics
- File server access (SMB/NTFS)
- PowerShell scripting
- Network documentation and diagrams


## Repository Contents


### **1. PowerShell Scripts**
- `get-system-info.ps1`
- Gathers system info including network configuration, OS version, and uptime.


### **2. Configuration Samples**
- `sample-dhcp-scope-config.txt`
- Example DHCP scope layout used in enterprise environments.


### **3. SysAdmin Documentation**
- `ipv4-dns-dhcp-troubleshooting.md`
- `windows-endpoint-troubleshooting.md`
- `file-server-access-guide.md`


### **4. Network Diagrams**
- `network-map.md`
